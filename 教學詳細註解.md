## 1. 專案分層架構總覽

本專案後端採用五層分層設計，讓程式碼更好維護、測試、擴充。

- **Contracts**：介面層，只放規格（如 IMemberService、IQuestionAnswerService）
- **Api**：控制器層，只負責接收 HTTP 請求、回應資料
- **Application**：業務邏輯層，處理驗證、流程、規則
- **Implementation**：實作層，放真實服務（連資料庫/AI）和 Mock 服務（假資料/假AI）
- **Models**：資料結構層，只放資料結構（如 Member）

## 2. Mock/真實服務切換教學

### 什麼是 Mock？
- Mock 就是假資料或假服務，讓你不用連資料庫或外部AI，也能測試整個系統。
- 本專案支援「會員」與「AI 問答」都可 Mock。

### 怎麼切換？
1. 打開 `ChatDemo/Program.cs`
2. 找到：
   ```csharp
   bool useMock = true; // true=用假資料，false=用真實服務
   ```
3. `true` 代表用 Mock（假資料），`false` 代表用真實服務
4. 存檔後重新啟動服務即可切換

### Mock 會員服務
- 內建兩個假會員（testuser、alice）
- 註冊、登入、查詢帳號都只在記憶體，不會寫進資料庫

### Mock AI 問答服務
- 只會回傳「[Mock 回覆] 你問了：xxx，這是測試用的 AI 回答。」

## 3. Program.cs DI 註冊切換範例

```csharp
// Program.cs 片段
bool useMock = true; // ← 只要改這裡即可切換

if (useMock)
{
    builder.Services.AddScoped<IMemberService, MockMemberService>();
    builder.Services.AddScoped<IQuestionAnswerService, MockQuestionAnswerService>();
    builder.Services.AddScoped<IDatabaseService, DapperDatabaseService>(); // 若有用到還是要註冊
}
else
{
    builder.Services.AddScoped<IDatabaseService, DapperDatabaseService>();
    builder.Services.AddScoped<IMemberService, DapperMemberService>();
    builder.Services.AddScoped<IQuestionAnswerService, OllamaService>();
}
```

## 4. 每層範例（含AI Mock）

### Contracts（介面層）
```csharp
/// <summary>會員服務介面</summary>
public interface IMemberService {
    Task<bool> RegisterAsync(Member member);
    Task<Member?> LoginAsync(string username, string password);
    Task<bool> UsernameExistsAsync(string username);
}

/// <summary>AI 問答服務介面</summary>
public interface IQuestionAnswerService {
    Task<string> GetAnswerAsync(string prompt);
}
```

### Api（控制器層）
```csharp
/// <summary>會員API</summary>
[ApiController]
[Route("api/[controller]")]
public class MemberController : ControllerBase {
    private readonly MemberApplicationService _memberApplicationService;
    [HttpPost("register")]
    public async Task<IActionResult> Register([FromBody] Member member) {
        var result = await _memberApplicationService.RegisterMemberAsync(member);
        return Ok(result);
    }
}

/// <summary>AI 問答API</summary>
[ApiController]
[Route("api/[controller]")]
public class OllamaController : ControllerBase {
    private readonly QuestionAnswerApplicationService _qaService;
    [HttpGet("question/{question}")]
    public async Task<IActionResult> GetAnswer(string question) {
        var result = await _qaService.GetAnswerAsync(question);
        return Ok(result);
    }
}
```

### Application（業務邏輯層）
```csharp
/// <summary>會員應用服務</summary>
public class MemberApplicationService {
    private readonly IMemberService _memberService;
    public async Task<ApplicationResult<bool>> RegisterMemberAsync(Member member) {
        if (string.IsNullOrWhiteSpace(member.Username))
            return ApplicationResult<bool>.Failure("用戶名不能為空");
        var result = await _memberService.RegisterAsync(member);
        return ApplicationResult<bool>.Success(result);
    }
}

/// <summary>AI 問答應用服務</summary>
public class QuestionAnswerApplicationService {
    private readonly IQuestionAnswerService _qaService;
    public async Task<ApplicationResult<string>> GetAnswerAsync(string question) {
        if (string.IsNullOrWhiteSpace(question))
            return ApplicationResult<string>.Failure("問題不能為空");
        var answer = await _qaService.GetAnswerAsync(question);
        return ApplicationResult<string>.Success(answer);
    }
}
```

### Implementation（實作層）
```csharp
/// <summary>Mock 會員服務</summary>
public class MockMemberService : IMemberService {
    private static readonly List<Member> _mockMembers = new() {
        new Member { Id = 1, Username = "testuser", Password = BCrypt.Net.BCrypt.HashPassword("123456"), Email = "testuser@example.com" },
        new Member { Id = 2, Username = "alice", Password = BCrypt.Net.BCrypt.HashPassword("alicepwd"), Email = "alice@example.com" }
    };
    public Task<bool> RegisterAsync(Member member) {
        if (_mockMembers.Any(m => m.Username == member.Username)) return Task.FromResult(false);
        member.Id = _mockMembers.Max(m => m.Id) + 1;
        member.Password = BCrypt.Net.BCrypt.HashPassword(member.Password);
        _mockMembers.Add(member);
        return Task.FromResult(true);
    }
    public Task<Member?> LoginAsync(string username, string password) {
        var member = _mockMembers.FirstOrDefault(m => m.Username == username);
        if (member != null && BCrypt.Net.BCrypt.Verify(password, member.Password))
            return Task.FromResult<Member?>(member);
        return Task.FromResult<Member?>(null);
    }
    public Task<bool> UsernameExistsAsync(string username) {
        return Task.FromResult(_mockMembers.Any(m => m.Username == username));
    }
}

/// <summary>Mock AI 問答服務</summary>
public class MockQuestionAnswerService : IQuestionAnswerService {
    public Task<string> GetAnswerAsync(string prompt) {
        return Task.FromResult($"[Mock 回覆] 你問了：{prompt}，這是測試用的 AI 回答。");
    }
}
```

## 5. 常見錯誤排查

- 問：Mock 會員資料會存到資料庫嗎？
  答：不會，只存在記憶體，重啟就消失。
- 問：Mock AI 會真的呼叫 AI 嗎？
  答：不會，只是回傳固定字串。
- 問：切換 Mock 沒有效果？
  答：記得存檔後重啟服務。
- 問：Swagger 沒看到 API？
  答：請確認 `dotnet run` 有啟動成功，網址 http://localhost:5006/swagger
- 問：要怎麼切回真實服務？
  答：把 `useMock` 改成 `false`，存檔重啟即可。

## 後端 Services/DapperDatabaseService.cs 完整註解

```csharp
using System.Data; // 匯入資料庫相關
using System.Data.SqlClient; // 匯入 SQL Server 連線
using Microsoft.Extensions.Configuration; // 匯入組態設定

namespace ChatDemo.Services
{
    // 資料庫服務介面
    public interface IDatabaseService
    {
        IDbConnection CreateConnection(); // 建立資料庫連線
    }
    
    // 資料庫服務實作
    public class DapperDatabaseService: IDatabaseService
    {
        private readonly string _connectionString; // 資料庫連線字串

        // 透過建構子注入組態，取得連線字串
        public DapperDatabaseService(IConfiguration configuration)
        {
            _connectionString = configuration.GetConnectionString("DefaultConnection");
        }

        // 建立 SQL Server 連線
        public IDbConnection CreateConnection()
        {
            return new SqlConnection(_connectionString);
        }
    }
} 
```
